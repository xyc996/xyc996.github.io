整体介绍

这个网站建立的目的是帮助新英才高中的学生学习信息技术课程。

内容​

主要内容有 Python 入门和 WPS 表格入门。 其中 Python 部分包括软件的准备和环境的配置。其实这些内容在官方文档已经有介绍，这里为了照顾一些同学没有 PC(个人电脑),附带介绍了在 Android 平台上编写和运行 Python 的方法。其实这些东西网络上大把的，只要动动手搜索一下都不是问题，同学们需要的是动手能力和坚持，最重要的是主动学习的意识。题

有必要学吗？我能学会吗？​

有些同学觉得自己没有学籍，不考试就没必要学了；还有一些同学认为自己以后用不上这些东西就没必要学了； 还有一些同学认为自己其他成绩差、学不会干脆不学了...... 首先明确：学习不全是为了考试，更不仅仅是为了谋生。学习知识能使我们对这个世界有更深入清晰的理解，让我们活得更加通透。更好的谋生手段只不过是这一行为的附加产物罢了。因此学习知识不能太过急功近利。当今世界发展迅速，谁又能确保你认为的‘能用到的东西’以后一定有用呢。就算从实用角度出发，你也一定听说过‘书到用时方恨少’，我们在学习阶段本来就需要储备大量的知识技能以备不时之需。 至于那些没有信心的同学，我想对你们说，要用发展的眼光看问题，其他学科差不代表这个学科学不好，过去成绩差不代表以后永远差下去。时间在变，人也在变，找到差的原因，逐步把遗漏的知识补上来，不就不差了吗？人的智力发展是不平衡的，有一些大器晚成的人就是开窍晚一些，还有一些人前期因为客观条件限制而表现不太理想，重要的是坚持向着正确的方向努力。 我想说只要你是个智力正常的人，这些东西都能学会、学好。可能你学习的过程中有一些不懂的地方、会遇到各种各样的问题，这都再正常不过了。多数优秀人都是这么过来的。只要你肯钻研，勤实践，多找有经验的人请教，利用一切能够自我提升的资源，坚持下去，相信星星之火终可以燎原。对于没有信心的人来说，每解决一个问题，每掌握一个技能都能增强你的信心。当你的知识越来越丰富，信心越来越充足的时候，不知不觉你不就变成一个有信心有知识的优秀的人了吗？ 因此学习这些知识很有必要，也能学会。附带的你也能更好地理解这个信息时代。

具体怎么学？​

根据你的情况参照这上面的指引和资料配置好环境，理解相应的概念，踏踏实实把基础语法弄懂练会。再去做一些案例，后期根据自己的需求写一些实用的程序，学以致用。 学的过程中遇到问题可借助豆包、deepseek 等人工智能平台来辅助解决疑问。也可给scrcpy@foxmail.com发邮件，描述清楚你的问题（附带你的班级姓名）。 对于一些连基本键盘输入都不熟练的同学，有电脑的下载一个类似于金山打字的程序，星期天回家对着练习。没有的打印一个键盘的纸。上网搜一个打字教程对着练习。必须解决输入的问题。
