准备工作和环境搭建

我们介绍过，Python 是一门解释型的编程语言，因此 python 程序在计算机上执行需要一个‘翻译’，就是解释器，来把你写的程序翻译成计算机能够执行的指令。

如果你有一台 PC(个人电脑)​

那么你可以按照官方文档的指引去下载对于版本的 python 并安装在你的机器上，然后照着教程去学习和练习。

如果你没有 PC​

你至少有一台安卓手机或平板电脑，你仍然可以通过一些微小的操作来在你的设备上练习 python。如果你在网上搜索或者在人工智能平台（如豆包）提问 “怎么在安卓上运行 python”，那么你会得到好几种可行的方案。如：

  • Pydroid 3 ：专为 Android 设备设计，支持完整的 Python 3 环境，有方便的图形用户界面。它支持安装多种 Python 库，如 NumPy、Pandas、Matplotlib 等，适合进行数据分析和科学计算，并且用户可在离线状态下编写和运行 Python 代码，方便随时随地练习。
  • QPython ：功能强大，支持 Python 2 和 Python 3 环境，提供集成开发环境（IDE），包括代码编辑器、控制台和脚本管理功能。用户可通过 QPython 编写、调试和运行 Python 脚本，使用内置的第三方库进行扩展，还支持通过插件扩展功能，如 Web 开发、图形界面开发等，适合有一定编程经验的用户。 <strong>你可以在手机/平板的应用商店下载这两个软件中的一个来进行练习</strong>

配套资源​

这里提供 PC 平台的 python 3.8.10 的安装包，安卓平台的 QPython 安装包，附带一个入门的学习教程，和中文版官方文档。大家可以把文档下载保存在自己设备上，随时查询和学习。 官方文档内容比较多，但是全面。附带的教程内容相对少一些，讲解详细易懂一些。建议先研究清楚简易教程，再对照官方文档作更深入的学习。
