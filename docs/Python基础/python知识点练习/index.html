第一部分：课程目标与热身​

1. 目标

  • 回顾Python核心语法与概念
  • 巩固程序三种基本结构：顺序、分支、循环
  • 熟练掌握函数的使用
  • 理解并会使用常见数据结构：列表、字典
  • 能够编写解决简单问题的代码

2. 热身思考：什么是程序？ 程序 = 数据结构 + 算法

  • 数据结构 ：如何把数据在计算机中组织、存储。例如：变量、列表、字典。
  • 算法 ：解决问题的具体步骤和流程。例如：先判断再计算，循环处理等。

第二部分：核心知识点回顾​

1. 变量与数据类型

  • 变量 ：数据的载体，如 name = "小明"
  • 基本数据类型 :
  • 整型 (int) : age = 18
  • 浮点型 (float) : height = 1.75
  • 字符串 (str) ：greeting = "Hello"
  • 布尔型 (bool) ：is_pass = True, is_rain = False

    print("Hello, World!") # 输出字符串

3. 运算符

  • 算术运算符 ：+, -, *, /, // (整除), % (取余), ** (幂)
  • 比较运算符 ：==, !=, >, <, >=, <= (返回值是Bool类型)
  • 逻辑运算符 ：and, or, not (操作Bool类型，返回值也是Bool类型)

4. 程序的三大控制结构

a. 顺序结构 程序默认从上到下逐行执行。

b. 分支结构 (if, elif, else)

python

score = int(input("请输入成绩："))
if score >= 90:
    print("优秀")
elif score >= 70: # 70 <= score < 90
    print("良好")
elif score >= 60:
    print("及格")
else:
    print("不及格，要加油了！")

c. 循环结构

  • for 循环 ：已知循环次数，遍历序列 python
  # 打印 0到4
  for i in range(5):
      print(i)

  # 遍历列表
  fruits = ["苹果", "香蕉", "橘子"]
  for fruit in fruits:
      print("我喜欢吃", fruit)
  • while 循环 ：条件满足就一直循环 python
  count = 0
  while count < 5:
      print("Count is", count)
      count += 1 # 千万别忘了修改条件变量，否则会死循环！

  # 另一个例子
  password = ""
  while password != "123456": # 当密码不是"123456"时，就一直循环
      password = input("请输入密码：")
  print("登录成功！")
  • 循环控制 :
  • break：立即跳出整个循环
  • continue：跳过本次循环的剩余语句，直接进入下一次循环

课堂练习3：猜数字游戏 编写一个猜数字的小游戏，数字为1-10之间的一个随机整数，给用户最多3次机会。

python

# 示例答案
import random

secret_num = random.randint(1, 10)
guess_count = 0
max_attempts = 3

while guess_count < max_attempts:
    guess = int(input("请猜一个1-10的数字："))
    guess_count += 1

    if guess == secret_num:
        print("恭喜你，猜对了！")
        break
    elif guess < secret_num:
        print("猜小了")
    else:
        print("猜大了")

    if guess_count == max_attempts:
        print("机会用完了，秘密数字是：", secret_num)

5. 数据结构

a. 列表 (List) - 有序的集合

  • 创建 : list1 = [1, 2, 3, 'a']
  • 索引 : list1[0] -> 1, list1[-1] -> 'a'
  • 切片 : list1[1:3] -> [2, 3]
  • 常用方法 :
  • .append(x) 在末尾添加元素
  • .pop() 删除末尾元素
  • .index(x) 查找x的索引
  • len(list1) 获取列表长度

b. 字典 (Dict) - 键值对的集合

  • 创建 : student = {'name': '张三', 'age': 18, 'score': 90}
  • 访问 : student['name'] -> '张三'
  • 常用方法 :
  • .keys() 获取所有键
  • .values() 获取所有值
  • .items() 获取所有键值对
  • dict.get(key, default) 安全获取值，如果key不存在返回default

6. 函数 (Function) - 代码的封装

  • 作用 ：复用代码，简化逻辑，模块化编程
  • 定义 : python
  def greet(name): # name是形参
      """这是一个问候函数（文档字符串）"""
      print("Hello,", name)
      return None # 默认返回None

  # 调用函数
  greet("World") # "World"是实参
  • 带返回值的函数 : python
  def add(a, b):
      result = a + b
      return result # 使用return返回结果

  sum_result = add(5, 3)
  print(sum_result) # 输出 8

课堂练习5：编写函数 编写一个函数 is_even(num)，判断一个数字是否为偶数，返回布尔值。

python

# 示例答案
def is_even(num):
    if num % 2 == 0:
        return True
    else:
        return False
    # 或者更简洁的写法: return num % 2 == 0

print(is_even(4)) # True
print(is_even(5)) # False

第三部分：综合实战​

题目：简易学生成绩管理系统 请编写一个程序，实现以下功能：

  1. 提供一个循环操作菜单： text

    1. 添加学生成绩
    2. 查看所有学生成绩
    3. 查询平均分
    4. 退出

  2. 选择 1，可以输入学生姓名和成绩，并存储起来。

  3. 选择 2，可以打印出所有学生的姓名和成绩。

  4. 选择 3，可以计算并打印出所有学生的平均分。

  5. 选择 4，退出程序。

(思考时间，然后老师带领一起实现)

python

# 示例答案框架
students = [] # 用一个空列表存储所有学生数据（字典）

while True:
    print("\n--- 学生成绩管理系统 ---")
    print("1. 添加学生成绩")
    print("2. 查看所有学生成绩")
    print("3. 查询平均分")
    print("4. 退出")

    choice = input("请选择操作 (1/2/3/4): ")

    if choice == '1':
        name = input("请输入学生姓名: ")
        score = float(input("请输入学生成绩: "))
        student = {"name": name, "score": score}
        students.append(student)
        print("添加成功！")

    elif choice == '2':
        if len(students) == 0:
            print("暂无学生数据！")
        else:
            print("\n姓名\t成绩")
            for s in students:
                print(f"{s['name']}\t{s['score']}")

    elif choice == '3':
        if len(students) == 0:
            print("暂无学生数据，无法计算平均分！")
        else:
            total = 0
            for s in students:
                total += s['score']
            average = total / len(students)
            print(f"平均分为: {average:.2f}") # :.2f 表示保留两位小数

    elif choice == '4':
        print("感谢使用，再见！")
        break

    else:
        print("输入有误，请重新选择！")

第四部分：总结与备考建议​

1. 知识体系总结

  • 基础 ：变量、数据类型、输入输出
  • 结构 ：顺序、分支(if)、循环(for/while)
  • 数据 ：列表(list)、字典(dict)
  • 封装 ：函数(def)

2. 编程思维培养

  • 拆解问题 ：将大问题分解成多个小步骤
  • 模式识别 ：发现重复操作，用循环解决
  • 抽象 ：使用函数和数据结构来组织代码

3. 备考与练习建议

  • 多读代码 ：理解别人的编程思路。
  • 多敲代码 ：亲自动手调试，解决错误（Bug）是最好的学习。
  • 刷经典题 ：如斐波那契数列、判断质数、排序、查找等。
  • 注意细节 ：缩进、冒号、英文符号、拼写错误是常见失分点。
